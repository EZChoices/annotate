# Annotator Guide

**Purpose:** This guide explains how annotators will work with the Dialect Data platform.

### Steps:
1. Open the annotation web app on your phone.
2. Listen to the clip.
3. Verify or correct transcript.
4. Tag:
   - Dialect
   - Gender
   - Speaker count
   - Background noise level
5. Save and go to next clip.

**Important:** Annotators will NOT download files. All audio stays hosted securely on Dialect Data servers.
